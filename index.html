Semantic Segmentation

Semantic segmentation,is the attempt to understand an image on a per pixel basis. This is done by classifying the pixels of an image into a set of classes. Automating this process is an important process in computer vision based applications like autonomous vehicles, and real-time augmented reality. Here I worked to create a model capable of learning to output masks representing the hair contained within in an image, for application in real-time augmentation of hair color, or in photo editing applications.

The model itself was modeded after SegNet created by Badrinarayanan et al. While the original model was implemented in Caffe I used Tensorflow and Keras' Functional API to implement my version. The architecture of the model consists of first and Encoder and then a Decoder. Both of which are then connected as in a U-Net model. With the outputs of the downsampling units of the Encoder connecting directly to the input of the upsampling layers of the Decoder. This allows the model to directly upsample from the feature maps learned at each layer of the Encoder. For more detail on the structure of the model see the poster below or read "SegNet: A Deep Convolutional Encoder-Decoder Architecture for Image Segmentation" by Badrinarayanan et al.

Super-Resolution

Super-resolution is the process of upscaling a source image into a new image at a higher resolution. Super-resolution has applications in areas like facial recognition, medical imaging, surveilance, HD/4K video processing and more. In my example work based on the paper "Real-Time Single Image and Video Super-Resolution Using an Efficient Sub-Pixel Convolutional Neural Network" by Shi et al.

In thier paper Shi et al. construct a model that consists of three convolutional layers. The first two layers are traditional convolutional layers for learning feature maps. While the third layer acts as a sup-pixel convolutional layer that aggregates all the activated feature maps from the low resolution images. This aggregation is what then constructs the new higher resolution image. My implementation was build using Tensorflow and trained using Google Colab, with the model trained using the Berkely BSDDS500 dataset. The model produces HD images from low resolution images constructed from the original dataset.

Generative Adversial Networks

Generative Adversial Networks (GANs) are a state of the art machine learning framework. Which essential consists of two models a Generator and Discriminator. These sub-models are trained in a zero-sum game, until the Discriminator can no longer classify between real images, and ones created by the Generator. GANs have produced state of the art results in fields like: Deepfakes, Image synthesis, automatic data augmentation, image compression, Video Generation, Image Style-Transfer and more. This is my research presentation along with some sample code that uses a simple GAN to conduct Style-Transfer between to images.

Quantum Computing

Quantum Computing is a new approach to computation that breaks away from the traditional computer architectures of today, like Von Nueman and Harvard. This is accomplished by leveraging Quantam Mechanics and the spin of an electron or the polarization of a photon to store and represent data as Qubits (Qbits), or Quantum bits. To understand the difference in power between a traditional comptuer comptuer and a Quantum one, we first need to look at regular bits vs Qbits.Traditional binary bits can only ever hold one of two possible values 1 or 0. This means that for any number of N bits, a system can represent one of the 2N possible states at any given time. A Qbit on the other hand can not only represent either a 1 or 0 discretely like a traditional bit. It also allows us to represent a superposition of both states that is, both 1 and 0 at once (just like Schrodinger's cat). Meaning that for any N sized Qbit system we can simultaneously represent all 2N possible states. Allowing Quantam Computers to in theory to solve complex problems that would take a normal computer billions of years to calculate.

One such example of a Quantum Computation breaking the limitations of traditional computers is in cryptography. In 1994 computer sceintist at MIT Peter Shor found an algorithm for finding the prime factor of numbers using a Quantum Computer. Shor's algorhtim would in theory allow a 4,000 or so Qbit system to crack today's RSA encryption in a matter of seconds. While today's largest supercomputer would take some 300 trillion years for the same encryption.

Custom Programing Language

Learn is my own programing language. Designed as part of a project to better understand Programing Language Theory. Learn is meant to be as straight forward as its name. Learn is an educational programming language that focuses on a learn by doing approach. In keeping with this ideal: simplicity, orthogonality, writability, readability, and expressivity will be of the utmost concern. With the goal of not overwhelming users with options and a complex web of interrelated types and structures. While at the same time still providing the core fundamentals needed in any language, and enough orthogonality to allow the programmer to handle a large set of problems.
